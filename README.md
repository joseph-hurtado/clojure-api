# Clojure API

A barebones Clojure REST API, for learning purposes.

## Dependencies

Make sure you have Clojure installed.  Also, install the [Heroku Toolbelt](https://toolbelt.heroku.com/).

## Running Locally

```
lein run
```

## Deploying to Heroku

```sh
$ heroku create
$ git push heroku master
$ heroku open
```
